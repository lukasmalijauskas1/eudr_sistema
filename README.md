# eudr_sistema
Įdiegti XAMPP ir paleisti „Apache“ bei „MySQL“ servisus.
Projekto katalogą „EUDR_SISTEMA“ nukopijuoti į "C:\xampp\htdocs". Šiame kataloge yra visi PHP ir CSS failai, bibliotekos „TCPDF“ ir „tFPDF“ bei duomenų bazės struktūros failas eudr_sistema.sql.
„phpMyAdmin“ aplikacijoje sukurti naują duomenų bazę „eudr_sistema“ ir į ją importuoti failą eudr_sistema.sql, esantį projekto kataloge.
Naršyklėje atidaryti adresą http://localhost/EUDR_SISTEMA/login.php ir prisijungti administratoriaus paskyra. Sėkmingas prisijungimas patvirtina, kad žiniatinklio serveris, duomenų bazė ir aplikacija įdiegti teisingai ir sistema parengta naudojimui.
